# Veerge

This is a real estate management app

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Or, run the unit tests:

```bash
npm run test
# or
yarn test
# or
pnpm test
# or
bun test
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- src: Contains app source code i.e ./src
- app: This is the router file used in next i.e ./app
- assets: contains reusable assests i.e ./assets
- components: contains components to be used all through app i.e ./components
- constants: contains constants to be used in app i.e ./constants
- modules: contains application modules e.g dashboard, listing, users, etc. i.e ./modules
- utils: conatins general utility functions i.e ./utils
- Config files for next, tailwind, vitest, eslint, etc. exist in top directory i.e ./
